# novel_proofreader

VSCode と textlint とかで小説の校正するやつ

## 使い方

1. `npm i`します。
2. VSCode に[textlint の拡張機能](https://marketplace.visualstudio.com/items?itemName=taichi.vscode-textlint)をインストールします。
3. マークダウンファイル(.md)を作り、そこに小説を書きます。
4. うまい！
